Formas de usar CSS

  1. CSS Inline

    • Como funciona: Aplica estilos diretamente em elementos HTML usando o atributo style.

    Vantagens

    • Fácil e rápido para aplicar estilos em elementos individuais.
    • Útil para testes rápidos e visualização de mudanças.
    • Não requer um arquivo CSS separado.

    Desvantagens

    • Torna o código HTML mais longo e menos organizado.
    • Difícil de manter e atualizar, especialmente em sites grandes.
    • Não promove a separação de conteúdo e apresentação.

  2. CSS Interno (Incorporado):

    • Como funciona: Define estilos dentro da tag <style> na seção <head> do documento HTML.

    Vantagens

    • Útil para aplicar estilos a uma única página web.
    • Boa opção para projetos menores onde a organização não é crítica.

    Desvantagens

    • Não promove a reutilização de estilos em várias páginas.
    • Pode tornar o código HTML da página mais longo e menos legível.

  3. CSS Externo:

    • Como funciona: Define estilos em um arquivo separado com extensão .css
      e o vincula ao documento HTML usando a tag <link>.

    Vantagens

    • Promove a separação de conteúdo e apresentação, tornando o código mais limpo e organizado.
    • Permite a reutilização de estilos em várias páginas, facilitando a manutenção e atualização do site.
    • Ideal para projetos maiores e sites complexos.

    Desvantagens

    • Requer um arquivo CSS separado, o que pode adicionar complexidade ao projeto.
    • A vinculação do arquivo CSS externo pode aumentar o tempo de carregamento da página se não for otimizada.
